# SauceDemo Playwright Automation

This project contains Playwright automated tests for https://www.saucedemo.com/.

## Requirements
- Node.js (>=14)
- npm

```

## Run tests
Run all tests (headless):
```bash
npm test
```

Run tests with browser visible (headed):
```bash
npm run test:headed
```

Open HTML report after test run:
```bash
npm run report
```
